3과목 : 데이터 분석 기획

3_2 :: 분석 마스터 플랜

3_2_1_1 :: 마스터 플랜 수립

 - 우선순위 고려요소 :: 전략적 중요도, 비즈니스 성과(ROI), 실행 용이성
 - 적용범위/방식 :: 업무내재화 적용수준, 분석데이터 적용수준, 기술적용수준

3_2_1_2 :: 수행 과제 도출 및 우선순위 평가

 - 과제우선순위 평가기준 :: 전략적 중요도, 실행용이성
 - 전략적 중요도 :: 전략적 필요성, 시급성
 - 실행 용이성 :: 투자 용이성, 기술 용이성

 - 3V : 투자비용 요소
 - Value : 비즈니스 효과
 - 시급성(Value), 난이도(3V)

3_2_2_1 :: 분석 거버넌스 체계 수립
분석기획 및 관리 수행 조직,  과제 기획 및 운영 프로세스, 분석 관련 시스템, 
데이터, 분석 관련 교육 및 마인드 육성 체계

3_2_2_2 :: 데이터 분석 성숙도 모델 및 수준진단

⁯분석준비도
BIC Data OS
 - Business, Infra, Cuture, Data, Organization, Skill

분석성숙도 :: 도입 > 활용 > 확산 > 최적화

분석성숙도 모델 :: CMMI
도입단계 :: 분석을시작하여 환경과 시스템을 구축
활용단계 :: 분석 결과를 실제 업무에 적용
확산단계 :: 전사 차원에서 분석을 관리 공유
최적화단계 :: 분석을 진화시켜 혁신 및 성과향상에 기여

준비형 :: 낮은 준비도, 낮은 성숙도
정착형 :: 낮은 준비도, 내부에서 제한적으로 사용, 우선 정착 필요
도입형 :: 높은 준비도, 낮은 성숙도
확산형 :: 지속적인 확산이 가능

3_2_2_3 :: 분석 지원 인프라 방안 수립

3_2_2_4 :: 데이터 거버넌스 체계 수립
 - 전사 차원의 모든 데이터에 대해 정책 및 지침, 표준화, 운영조직 및 책임 등의 표준화된 관리 체계 수립
 - 프레임워크 및 저장소 구축
 - 마스터 데이터, 메타 데이터, 데이터 사전
 - 구성요소 :: 원칙, 조직, 프로세스

 - 데이터 표준화 
  - 데이터 표준 용어 설정, 명명규칙 수립, 메타 데이터 구축, 데이터 사전 구축 등 
  - 데이터 사전 :: 데이터 구조 체계, 메타 엔터티 관계 다이어그램 등 제공

 - 데이터 관리 체계
  - 데이터 생명 주기 관리 방안

 - 데이터 저장소 관리

 - 표준화 관리


3_2_2_5 :: 데이터 조직 및 인력방안 수립

데이터 분석 조직

조직구조 / 인력구성

집중형 / 기능형 / 분산형

3_2_2_6 :: 분석 과제 관리 프로세스 수립
